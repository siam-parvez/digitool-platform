# DigitTool Platform

> A modern digital tools marketplace built for fast browsing, simple cart management, and a clean product showcase experience.

## Overview

DigitTool Platform is a responsive React application that presents premium digital tools in a polished storefront-style layout. It combines a strong hero section, product browsing, cart interaction, pricing plans, and a guided onboarding flow to create a smooth user experience.

## Technology Stack

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- DaisyUI
- React Toastify
- Lucide React
- JSON data for products

## Features

1. Product browsing with a clean grid layout and a dedicated cart view.
2. Responsive landing page sections including hero, stats, steps, pricing, and CTA.
3. Smooth UI feedback with toast notifications and modern component styling.
